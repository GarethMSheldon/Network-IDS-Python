NETWORK INTRUSION DETECTION SYSTEM

A real-time, signature-based NIDS with GUI - Battlestar Galactica Theme

For educational and defensive cybersecurity use only. Not a replacement for enterprise-grade NIDS like Suricata or Zeek.

CYLON DETECTION FEATURES

Real-time TCP Monitoring

Monitor established & listening TCP connections in real-time across Windows systems.

Signature-based Detection

Detect port scans, brute-force attempts, and connections to known malicious ports.

Live Statistics Dashboard

Total connections, unique IPs, top ports, connection rate, and runtime counter.

Alert Logging

Severity levels (HIGH/MEDIUM/LOW) with exportable reports and persistent log files.

Dark-themed Terminal Log

Battlestar-style dark terminal interface for monitoring all activity.

In-memory Execution

Runs entirely in-memory with no external dependencies.

DECK REQUIREMENTS

Operating System

Windows 10/11 or Windows Server 2016+

PowerShell Version

PowerShell 5.1 or later

Privileges

Administrator rights required (script includes #Requires -RunAsAdministrator)

LAUNCH PROCEDURE

To activate the Network Intrusion Detection System:

Clone or download this repository.
Open PowerShell as Administrator.
Navigate to the project folder:
cd path\to\network-ids-gui
Execute the launch script:
.\Start-NIDS.ps1

DEFENSIVE CYBERSECURITY NOTE

This tool is designed for defensive cybersecurity purposes to help identify potential threats on your network. Use responsibly and only on networks you own or have permission to monitor.
